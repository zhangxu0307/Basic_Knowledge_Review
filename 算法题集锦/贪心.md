

## greedy 贪心法

### Jump Game

```
class Solution(object):
    def canJump(self, nums):
        """
        :type nums: List[int]
        :rtype: bool
        """

        maxState = nums[0]
        for i in range(1, len(nums)):
            if maxState >= i:
                maxState = max(nums[i]+i, maxState)
        return True if maxState >= len(nums)-1 else False
```
note：
1. 每一步探索最远能到的位置即可
2. 注意，这里数组元素代表的最远可以往前跳多远，而不是只能跳到这么远
